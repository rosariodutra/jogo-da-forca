# 🎮 Jogo da Forca

Jogo da forca clássico no terminal desenvolvido em Python, com categorias temáticas, níveis de dificuldade e placar de partidas.

## 📸 Preview

```

       -----
       |   |
       O   |
      /|\  |
      / \  |
           |
    =========

  Palavra: a l g o r i t m o
  Erros  : e, u
  Chances: 2 restante(s)

  💡 Dica: Sequência de passos para resolver um problema
```

## ✨ Funcionalidades

- 3 categorias temáticas: **Tecnologia**, **Análise de Dados** e **Finanças**
- 3 níveis de dificuldade: Fácil (com dica), Médio e Difícil
- Forca animada com 7 estágios
- Placar de vitórias e derrotas por sessão
- Interface colorida no terminal

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/rosariodutra/jogo-da-forca.git

# Entre na pasta
cd jogo-da-forca

# Execute
python forca.py
```

> Requer Python 3.6+. Sem dependências externas.
