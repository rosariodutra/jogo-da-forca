# 🎮 Jogo da Forca

> Jogo da forca clássico no terminal com categorias temáticas da área de dados, níveis de dificuldade e placar de partidas. (Desenvolvido em Python).

---

## 🧩 Contexto

Projeto desenvolvido para consolidar conceitos de lógica de programação, estruturas de controle, funções e manipulação de strings em Python. As categorias foram escolhidas propositalmente dentro do universo de dados e finanças — reforçando vocabulário técnico da área.

---

## 🎯 Funcionalidades

- 3 categorias temáticas: **Tecnologia**, **Análise de Dados** e **Finanças**
- 3 níveis de dificuldade: Fácil (com dica), Médio e Difícil
- Forca animada com 7 estágios
- Placar de vitórias e derrotas por sessão
- Interface colorida no terminal

## 🚀 Como executar

```bash
git clone https://github.com/rosariodutra/jogo-da-forca.git
cd jogo-da-forca
python forca.py
```

> Requer Python 3.6+. Sem dependências externas.
